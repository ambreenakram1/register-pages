 
### Cheat Sheet

## SAAS WATCHER

```bash
cd client-side
sass sass:.tmp --watch
```

## Tailwind Watcher
```bash
cd client-side
npx tailwindcss -i ./.tmp/main.css -o ./src/index.css --watch
```

 ## Run Express Server 
```bash
cd sever-side
nodemon index.js
```

## Run React App
```bash
cd client-side
yarn start
```
